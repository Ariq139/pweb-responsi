About Split Stream

Split Stream is a racing game project made by me using Epic Games's Unreal Engine 4. My goal is to complete it in 2020 or soon, make it a community-based suggestions/community submitted game, and release it on Steam.

Also, since i'm using UE, the minimum spec to play this game is probably a mid-high spec, like i3 processor or the equivalent, 4GB RAM, GTX 750 or something like that.

Below is the progress so far.
